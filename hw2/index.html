Section I: Bezier Curves and Surfaces

Part 1: Bezier curves with 1D de Casteljau subdivision

e Casteljau’s algorithm takes a set of points that define a Bezier curve and repeatedly uses linear interpolation to generate smaller sets of points until there is one point left. The final point lies on the Bezier curve. The parameter t that is used in de Casteljau’s algorithm is in the range [0, 1] and determines where on the Bezier curve the final point lies. If you have a two points p1 and p2, a new point, p3, can calculated with the equation p3 = ((1 - t) * p1) + (t * p2). evaluateStep uses de Casteljau’s algorithm to take a set of points and to execute one step of de Casteljau’s algorithm to the points, returning the newly updated points. As you can see, each new point lies somewhere on the line that connects the other two points, and if t is 0 or 1, then the new point is directly on one of the old points.
